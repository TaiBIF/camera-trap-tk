FAQ
---------
Q: 改資料夾名稱的話，再更新，資料庫那邊會發生什麼事?

A: 目前單機版的機制是，第一次匯入後就跟原資料夾沒關係了，照片都是用 thumbnails 資料夾裡的縮圖。所以他改原資料夾名稱，不管單機還是線上版都不會有影響。
